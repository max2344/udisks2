The /org/freedesktop/UDisks2/mdraid/* objects

Objects with object paths starting with /org/freedesktop/UDisks2/mdraid/ represent Linux Software RAID arrays. Such objects implement the org.freedesktop.UDisks2.MDRaid D-Bus interface.

These objects should not to be confused with block device objects (which are used for low-level block devices the OS knows about).
